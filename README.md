### Simple web application to create, update notes.

The user creates, updates the note in the modal window (Bootstrap 5 modal plugin).

The application is created using:
- PHP
- Laravel 10.x framework
- MySQL
- JavaScript
- Vue JS 3.x
- Inertia JS
- Bootstrap CSS 5.x
- HTML
